Standard metagenomic workflow

Configurable steps. The agent adapts tools to your data and resources.

QC

Host depletion

Assembly

Binning
MetaBAT/DAS Tool; checkM/GTDB-Tk for bin quality and taxonomy
5
Taxonomic profiling

6
Functional annotation
KEGG/eggNOG/GO pathways; module coverage summary
