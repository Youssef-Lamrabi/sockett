D) ROUTING & INSTRUCTIONS (agent/v2/utils/instructions.py)
--------------------------------------------------------------------------------
  - Règles d'intention (l'agent sait QUAND utiliser chaque nouvel outil sans qu'on
    le nomme) : Nanopore->flye, décontamination hôte, couverture->compute_coverage,
    ANI->fastani, arbre phylo->mafft/trimal/fasttree, SRA->fetch_sra_reads.
  - iss : formule de profondeur non-ambiguë (n_reads = depth*genome_bp/read_len,
          ne pas diviser par 2) + vérification post-simulation.
  - Prodigal : -p single pour isolat ; même mode des deux côtés pour comparer.
  - ncbi-genome-download : auto-récupération du cache assembly_summary corrompu
          ("Invalid line length ... Expected 38, got 39") : filtrer les lignes
          malformées puis réessayer (au lieu de tout abandonner).
  - ANI matrix : exclure la self-comparison (diagonale ~100%) avant de choisir la
          référence la plus proche (corrige "isolat = SMS-3-5 à 100%").
  - samtools sort : instruction alignée sur la syntaxe moderne 'sort -o out.bam
          in.bam' (l'ancienne doc positionnelle 0.x faisait échouer le step).
  - MAFFT : --anysymbol pour les résidus non-standard ('U') + wrapper par-cluster
          (un échec sauté, pas d'abandon global).
